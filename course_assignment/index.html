This website is designed to dynamically fetch and display weather data in a user-friendly manner. Users can view the latest weather measurements fetched from a backend API, presented in a table format with information such as measurement time, type, and value. The tables are ordered from oldest to newest measurement.

Users gain access to additional functionality, such as separate columns for date and time information, descriptive column headers, and bar chart visualizations of the data. They can also switch between different views, such as a default view displaying the latest 30 measurements, a view displaying the latest 20 temperature measurements, and a view displaying the latest 20 wind speed measurements.

Advanced features include drop-down menus that allow users to select a time interval for consideration, with corresponding data shown in the tables and visualizations. Users can also select a desired measurement type for viewing in a table and line chart with time increasing towards the right edge.

The website is designed to be visually appealing, with efficient use of screen space and an optional background color selection. Additionally, the Info view contains author information and a short description of the website's functionality.

Overall, this website provides an easy-to-use interface for accessing and understanding weather data, with increasing levels of functionality.

Extra Feature:
I implemented a selector option with main buttons, which allows the user to change the background color. The selector includes a list of twelve light colors. Since the main theme of the web page is green, only light background colors were chosen to maintain the theme. This helps to keep the main theme consistent while still allowing the user to customize the background to their preference.
